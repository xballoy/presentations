# Opaque Types

> Branded / opaque types in TypeScript for safer domain modeling: how to build them, why they help, and how they compose with validation libraries such as Zod and Valibot.

View the presentation online:

- [English version](https://xballoy.github.io/presentations/opaque-types/en/)
- [French version](https://xballoy.github.io/presentations/opaque-types/fr/)

## Development

From the repository root:

- `pnpm install`
- `pnpm dev:opaque-types`
- visit [http://localhost:3030](http://localhost:3030)

`pnpm dev:opaque-types` opens the English deck. To open a specific language directly:

- `pnpm --filter opaque-types dev:en`
- `pnpm --filter opaque-types dev:fr`

The demo lives in [`demo`](./demo). Run it with `npm install && npm test`.
